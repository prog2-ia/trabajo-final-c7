# Este proyecto es un simulador de inversión financiera desarrollado en Python.  
## Requisitos: 
- Python3
- No requiere librerias externas
  
Para usar el programa hay que ejecutar desde python el archivo: "main.py"

## Este proyecto es un simulador interactivo de inversión, desarrollado en Python, que permite a los usuarios:
- Registrarse e iniciar sesión
- Comprar y vender activos (Acciones, Fondos y Criptomonedas)
- Consultar su saldo
- Revisar su historial de transacciones
- Cargar activos desde archivo
- Guardar usuarios y sus datos automáticamente
